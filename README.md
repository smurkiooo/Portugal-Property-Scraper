Проект по скрэйпингу данных о недвижимости в Португалии. (При небольшой доработке подходит еще для Испании и Италии)

1) Был инициализирован проект с помощью пакетного менеджера uv. 

2) Былло инициализировано DVC окружение, использующее локальную папку в качестве "удаленного хранилища" для хранения картинок самой недвижимости.

**Описание проекта:**

 Для сбора данных о недвижимости была выбрана площадка Idealista.com в связи с тем, что данный сайт на запросы выдает статический HTML (а не генерирует JS динамически), который относительно удобно парсить.

Задача состоит в том, чтобы за адекватное время получать от сайта три вида артефактов:

 - структурированные данные (цена / этаж / площадь / координаты)
 - текстовое описание объекта недвижимости
 - изображения (будем считать, что это картинки, сопровождающие объявление)
